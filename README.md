<h1>Apna Dhaba</h1>

<h2>Description</h2>
Project consist of static landing webpage, which was designed for local dhabas situated mostly in villages, to sell their food items online to nearby persons. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML</b> 
- <b>CSS</b>
- <b>JavaScript</b>
